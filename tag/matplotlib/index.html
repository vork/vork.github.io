farrow-and-ball

A set of British paint manufacturer Farrow & Ball inspired color maps for matplotlib. The library is also available on [pypi](https://pypi.org/project/farrow-and-ball/). The maps are sorted in categories for different plotting needs (Divergent, Spectral, Base Colors, Misc).
